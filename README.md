# Urban Routes – Selenium Automation Testing Project

This project focuses on automated UI testing practices using Python, Selenium WebDriver, and Page Object Model (POM).

The project demonstrates scalable automation architecture, explicit waits, assertions, automated workflows, and structured UI validation using industry-standard QA automation practices.

# Project Scope

The project was designed to automate critical user flows of the Urban Routes application while applying scalable and maintainable automation testing practices.

Automation activities included:

- UI Automation Testing
- Selenium WebDriver Automation
- Page Object Model (POM)
- Explicit Waits
- Assertions Validation
- Automated User Flows
- End-to-End Scenario Validation
- Automation Architecture Organization
